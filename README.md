# 💈 Landing Page Barbearia

## 📖 Sobre o Projeto

Landing page desenvolvida para uma barbearia utilizando React e Next.js, com foco em design moderno, responsividade e experiência do usuário.

---

## ⚙️ Funcionalidades

- 💻 Interface moderna
- 📱 Layout responsivo
- ⚡ Navegação otimizada
- 🖼️ Sessões informativas
- 🎨 Design focado em conversão

---

## 🛠️ Tecnologias Utilizadas

- React
- Next.js
- JavaScript
- HTML5
- CSS3

---

## 🎯 Objetivo

Projeto desenvolvido para praticar desenvolvimento Front-end utilizando React e Next.js na criação de landing pages modernas e responsivas.
